資料庫無法串出並呈現於頁面的可能原因

By Linnyii 28 June 2021

在實作Work1_資料庫應用（新增/刪除/修改)的過程中，發現了以下幾種資料庫無法成功串出頁面的幾種可能原因：

忘記匯入JDBC Driver ：

導致資料庫conn 物件一直為null。

資料庫裡的資料可能不完整：

以自身情況舉例，寫了一個function 去列印出特定幾個欄位裡的資料，但遇到資料庫裡某些欄位為空時，則會拋出exception.

解決辦法為，將資料庫裡的資料補齊，或是另外撰寫邏輯判斷此情況應如何反應。

JSP file 裡所設定的Servlet 錯誤：

負責接收資料的Servlet 名稱必須正確或是真的存在，否則無法順利接收指令。

JSP file 所儲存的位置：

記得JSP file 必須存放於webapp 資料夾下才可以順利運作。

Servlet 接收後段資料後所要response 出去的JSP file 必須設定正確：

否則資料可能會回傳至錯的頁面，導致使用者看不到回傳的資料。
