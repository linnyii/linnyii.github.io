Work1_資料庫應用（新增/刪除/修改）

By Linnyii 28 June 2021
作品描述：

此為於Hahow_資料庫設計課程所要求之作業，

創造出一畫面連接資料庫，且擁有新增，刪除，修改等功能，

以下為大致做法描述

Step1:

藉由bootstrap工具快速提供頁面的表格，輸入框等呈現畫面。

Step2:

準備3個jsp file ，分別為Insert.jsp /gotoUpdate.jsp / gotoDelete.jsp

負責接收用戶所傳送的資料與呈現頁面結果給用戶。

Step3:

準備3個Servlet，分別為InsertEmployee /DeleteEmployee/UpdateEmployee

Servlet 作於接收jsp 前端頁面接收資料，並傳送給後端處理。

後端出裡完畢後，會接收後段回傳資料，並forward 給前端jsp頁面。

Step4:

準備一Service class ，用來第一步接收Servlet 所傳送之任務，以及用來收集或是彙整

後端所處理完的資料結果。

Step5:

準備一EmployeeDao class，主要實際執行資料的存取，，也是實際和資料庫建立連線的

class.，例如建立物件去執行sql 語法，或是將於資料庫所搜集之資料存放於物件，

並回傳出去。

Step6:

準備另一class，主要用來與資料庫建立連線。

使用工具：

JAVA /JDBC Driver/ Tomcate /Bootsrap /MySQL /WorkBench/Eclipse /JSTL

上課心得:

上完Java 0-1 與資料庫設計兩個課程後，終於對於架設網站系統有了初步完整的概念，

另外在完成作業的過程中（如增加刪除與修改的兩種功能），也對於MVC架構還有資料

庫之間如何串接有了更完整的理解。
